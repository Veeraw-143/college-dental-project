## ⚡ QUICK START (60 seconds)

1. **Install Ollama**
   - Download: https://ollama.ai
   - Install and restart computer

2. **Download Model**
   ```powershell
   ollama pull mistral
   ```
   (First time: ~5-10 minutes)

3. **Start Server** (already running, but if needed)
   ```powershell
   cd d:\college-dental-project
   .\env\Scripts\Activate.ps1
   python manage.py runserver
   ```

4. **Test Chatbot**
   - Open: http://localhost:8000
   - Click chat bubble (bottom-right)
   - Type any question
   - Get AI response in 2-3 seconds!
