❓ FAQ

Q: Do I need to do anything manually?
A: No! Sync is automatic. Just use Django Admin normally.

Q: When do Excel files update?
A: Instantly when you save/modify/delete records (2-3 seconds).

Q: Where are the Excel files?
A: D:\college-dental-project\excel\

Q: What if something goes wrong?
A: Check admin panel or run excel_export_all manually.

Q: Can I disable auto-sync?
A: Yes! Go to Admin → Excel Sync Status → Uncheck "Sync Enabled"

Q: Can I import Excel back to database?
A: Yes! Use: python manage.py excel_import_all
